Sim-to-Real Bird’s Eye View Tracking Pipeline

Computer Vision · State Estimation · C++

C++OpenCV Eigen3PyTorch Kalman FilterWaymo Dataset June 2025 – Present

A real-time C++ pipeline that transforms a monocular camera feed into a top-down bird's eye view and tracks multiple objects across frames. Uses Inverse Perspective Mapping with pre-computed look-up tables for fast perspective correction, a fixed-size Eigen-based Kalman Filter for smooth multi-object tracking through occlusions, and full pinhole camera calibration validated against the Waymo Open Dataset.

Spatial Telemetry Engine

Retrieval-Augmented AV Logs · Vector DB · Python

FastAPIPostgreSQL pgvectorOllama SentenceTransformers nuScenesWaymo June 2025 – Present

A FastAPI + PostgreSQL backend that ingests raw autonomous vehicle sensor logs from nuScenes and Waymo, embeds spatial-temporal state vectors (camera extrinsics, ego kinematics, 3D bounding boxes) using an in-process SentenceTransformer, and indexes them with pgvector for fast similarity search. Designed for querying "what was happening near this location at this time" across large AV datasets.
